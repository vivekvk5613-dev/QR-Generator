# QR Code Generator

React se bana hua full-featured QR Code Generator.

## Features
- **URL** — Website links ke liye
- **Text** — Koi bhi text/message
- **WiFi** — Scan karke connect ho jao
- **Contact** — vCard format
- **UPI** — Indian payment QR (GPay/PhonePe)
- Color customization (QR color + background)
- Size options: 150px – 512px
- Error correction levels: L / M / Q / H
- PNG download & clipboard copy

## Kaise Chalayein

```bash
npm install
npm start
```

Browser mein `http://localhost:3000` khulega.

## Build Karna

```bash
npm run build
```

`build/` folder mein production files banen ge.

## Tech Stack
- React 18
- qrcode (npm package)
- Pure CSS (no UI library)
